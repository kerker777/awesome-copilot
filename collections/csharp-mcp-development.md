## Collection 使用方式

### C# MCP Server Expert

推薦使用

此 chat mode 提供建置 C# MCP 伺服器的專家指導。

此 chat mode 適用於：
- 建立新的 MCP 伺服器專案
- 實作 tools 和 prompts
- 偵錯協定問題
- 最佳化伺服器效能
- 學習 MCP 最佳實務

為了獲得最佳結果，請考慮：
- 使用 instruction 檔案為所有 Copilot 互動設定上下文
- 使用 prompt 產生初始專案結構
- 切換到專家 chat mode 以獲得詳細的實作協助
- 提供您需要的 tools 或功能的具體細節
