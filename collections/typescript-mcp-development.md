## Collection 使用方式

### TypeScript MCP Server Expert

推薦

此 chat mode 提供建置 TypeScript/Node.js MCP server 的專業指導。

此 chat mode 適用於：
- 使用 TypeScript 建立新的 MCP server 專案
- 使用 zod 驗證實作工具、資源和 prompts
- 設定 HTTP 或 stdio 傳輸
- 偵錯 schema 驗證和傳輸問題
- 學習 TypeScript MCP 最佳實務
- 最佳化 server 效能和可靠性

若要取得最佳結果，建議：
- 使用 instruction 檔案設定 TypeScript/Node.js 開發的上下文
- 使用 prompt 產生包含適當組態的初始專案結構
- 切換到專業 chat mode 以獲得詳細的實作協助
- 指定您是否需要 HTTP 或 stdio 傳輸
- 提供您需要的工具或功能的詳細資訊
