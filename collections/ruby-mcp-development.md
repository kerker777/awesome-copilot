## Collection 使用方式

### Ruby MCP Expert

推薦

此 chat mode 提供建置 Ruby MCP server 的專業指導。

此 chat mode 適用於：
- 使用 Ruby 建立新的 MCP server 專案
- 實作工具、prompts 和資源
- 設定 stdio 或 HTTP 傳輸
- 偵錯 schema 定義和錯誤處理
- 學習使用官方 SDK 的 Ruby MCP 最佳實務
- 與 Rails 應用程式整合

若要取得最佳結果，建議：
- 使用 instruction 檔案設定 Ruby MCP 開發的上下文
- 使用 prompt 產生初始專案結構
- 切換到專業 chat mode 以獲得詳細的實作協助
- 指定您是否需要 stdio 或 Rails 整合
- 提供您需要的工具或功能的詳細資訊
- 說明您是否需要身分驗證或 server_context 使用
