## Collection 使用方式

### Swift MCP Expert

推薦

此 chat mode 提供建置 Swift MCP server 的專業指導。

此 chat mode 適用於：
- 使用 Swift 建立新的 MCP server 專案
- 實作 async/await 模式和基於 actor 的並行
- 設定 stdio、HTTP 或網路傳輸
- 偵錯 Swift 並行和 ServiceLifecycle 整合
- 學習使用官方 SDK 的 Swift MCP 最佳實務
- 針對 iOS/macOS 平台最佳化 server 效能

若要取得最佳結果，建議：
- 使用 instruction 檔案設定 Swift MCP 開發的上下文
- 使用 prompt 產生初始專案結構
- 切換到專業 chat mode 以獲得詳細的實作協助
- 指定您是否需要 stdio、HTTP 或網路傳輸
- 提供您需要的工具或功能的詳細資訊
- 說明您是否需要資源、prompts 或特殊功能
